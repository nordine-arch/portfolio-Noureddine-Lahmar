Ingénierie pédagogique

Publication sur GitHub

  1. Modifiez les contenus ici.
  2. Cliquez sur Exporter content.json.
  3. Remplacez le fichier data/content.json dans votre dépôt GitHub.
  4. Ajoutez séparément les nouvelles images dans le dossier assets.

Cette page ne contient aucun mot de passe. Les écritures dans le dépôt restent protégées par votre compte GitHub.
