# Portfolio Noureddine Lahmar — GitHub Pages

Version statique du portfolio vert, sans avis ni notation par étoiles.

## Publication

Copiez **le contenu de ce dossier** à la racine du dépôt GitHub. Dans **Settings → Pages**, choisissez `Deploy from a branch`, branche `main`, dossier `/ (root)`.

## Administration

Ouvrez `admin.html`, modifiez les contenus, puis exportez `content.json`. Remplacez ensuite `data/content.json` dans le dépôt. Les nouvelles images doivent être ajoutées dans `assets/`.

La page d’administration est un éditeur statique : GitHub Pages ne peut pas enregistrer directement dans le dépôt sans exposer un jeton secret.
